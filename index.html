How to Build a PC

Breaking down the process of building a gaming PC into manageable steps is much less intimidating. Even if you're a newbie, don't worry: no previous experience building a PC is necessary.

PREPARATION 1: Tools to Build a PC

The first thing you'll need to do to prepare is gather the tools you need to complete the build. Prepare the materials described below in advance to ensure that there are no hiccups in the assembly process.Work space. You will need a large surface to work on, such as a table. To prevent accidental electrostatic discharge (which can damage sensitive components), make sure you are standing on a surface that is not carpeted. Screwdrivers. You'll need a #2 Phillips screwdriver for just about everything. Also, if you're installing an M.2 device, you'll need a #0 Philips screwdriver.

PREPARATION 2: Gaming PC Cases

Before you start choosing components, you should have a case, or at least a case size, in mind. The most important thing to consider when choosing a case is where you are going to put the computer. The final location of your PC will determine how big you can build it, and will also help you determine if some premium cases are worth buying. For example, you probably don't want to spend on a tempered glass side panel if the computer is going to be hidden under the desk. Cases typically come in three sizes: full tower, mid tower, and mini tower. These are very general categories (case sizes are not standardized), but they are based on the size of the motherboards.

PREP. 3: Gaming PC Parts

We definitely recommend that you prepare a budget before you start choosing components (it is very easy for the purchase to get out of control). Remember that you can always update each component later. STEP 1: INSTALL THE CPU

Parts/Tools: Motherboard, CPU Remove the motherboard from its antistatic packaging and place it on your work surface. Find the CPU socket, which will be covered by a protective plastic cap. On one corner of the plastic cap, or more commonly, on the socket itself, you'll see a small arrow. Record where the arrow is. Next to the CPU socket, you will see a small metal lever. Press the lever down and gently pull it to the side (away from the socket) to open the socket tray. Open the CPU and remove the packaging. Be very careful when handling the CPU. Both the CPU and the CPU socket are extremely susceptible to physical damage. Hold the CPU by the edges. Never touch the connectors on the bottom of the chip, because your fingers can add dust or oil, and try not to touch the top of the chip either.

STEP 2: (OPTIONAL) INSTALL M.2 SSD

Parts/Tools: Motherboard, M.2 SSD, Phillips #0 screwdriver, Motherboard User Manual. If you want to install an M.2 SSD, now is a good time to do so. First locate the M.2 slot on the motherboard. It's a small, horizontal slot with a tiny screw running through it. If you can't find it, if you find multiple M.2 slots, or if you plan to install more than one M.2 SSD, check the user manual that came with your motherboard. Remove the small screw with a Phillips #0 screwdriver. Don't lose it. Slide the M.2 SSD gently into the slot. When fully seated, it will protrude from the motherboard at an angle of about 35 degrees. Push the SSD down and replace the small screw to lock it in place.

STEP 3: INSTALL CPU COOLING

Parts/Tools: Motherboard with CPU installed, CPU cooler, Thermal paste, CPU cooler manual. There are different types of CPU coolers. For exact installation instructions, we recommend that you refer to the manual that came with your CPU cooler. Some coolers require a mounting bracket. The base plate might have a clamp pre-installed. You may need to remove this bracket if your refrigerator doesn't need it, or replace it if your refrigerator uses a different bracket. Do this before placing the motherboard inside the case.

STEP 4: INSTALL MEMORY (RAM)

Parts/Tools: Motherboard, RAM, Motherboard User Manual. Determine how many RAM slots the motherboard has (most have two or four). If you are going to use all available RAM slots, simply insert the RAM instead. If you are not going to use all of the RAM slots, check your user manual to find out what the correct setting is and use the RAM slots appropriately.

STEP 5: (OPTIONAL) PERFORM A TEST RUN OUTSIDE THE CABINET

Parts/Tools: Motherboard with CPU and CPU cooler installed, RAM, GPU, PSU, Screwdriver, Motherboard user manual, PC monitor (connected to GPU). Now that you've installed the CPU and CPU cooler, you might want to do a quick test run of the components to make sure they're all working. This test is much more difficult to perform (and harder to troubleshoot) once everything is installed in the chassis. To do this test, install the GPU and connect everything to the power supply (if you don't know how to install the GPU, see the next section). Make sure the power supply is connected to the motherboard (both the 8-pin and 24-pin of the CPU) and the GPU, then plug in and power on.

STEP 6: MOUNTING THE POWER SUPPLY

Parts/Tools: PSU, case, PSU cables, #2 Phillips screwdriver Unpack the PSU (or disconnect it from components if you chose to do a test run) and move the cables out of the way (if you can). Take a look at the case and figure out where the PSU should go (probably on the bottom near the back) and how it might be oriented. Ideally, you'll want to orient the PSU so that the fan is facing away from the case (through a vent). If the case has a bottom vent, you can mount the PSU upside down, as long as the bottom vent gets decent airflow when the PC is finished. If your case doesn't have vents, mount the PSU so that the fan is facing up (into the case) and make sure you have enough clearance.

STEP 7: INSTALL THE BASEPLATE

Parts/Tools: Case, motherboard, I/O shield (if not attached to motherboard), #2 Phillips screwdriver, screws, motherboard user manual. If your motherboard came with an unattached I/O shield (a rectangular sheet of metal with cutouts for the motherboard ports), you must first snap it into place on the back of the case (make sure the orientation is correct). correct). I/O shields have sharp edges, so watch your fingers. Once you've attached the I/O shield, you can install the motherboard. Double check to make sure all the cables are routed through the correct place, and then attach the motherboard (align it with the I/O shield first). If you're using a Phillips #2 screwdriver, install the first screw (the center screw) to hold the motherboard in place. Make sure not to drag the motherboard on the standoffs installed on the chassis.

STEP 8: INSTALL THE CPU

Parts/Tools: Motherboard, GPU, #2 Phillips screwdriver, screws, motherboard user manual. Locate the PCIe* x16 slot on the motherboard. It will be the longest PCIe* slot and may be colored differently than the others. If your motherboard has more than one PCIe* x16 slot, check your user manual to see if any slots need to be prioritized. If any slot can be used, determine which slot to use based on where the components are placed. It is recommended that the GPU has some space to ventilate. Depending on your case, it might be necessary to remove the I/O covers (small metal tabs that block the back panel of the case) to house the GPU I/O (HDMI, DisplayPort, DVI, etc.) and allow access from the outside of the chassis.

STEP 9: INSTALL THE STORAGE

Parts/Tools: Motherboard, SSD, HDD, #2 Phillips screwdriver, screws, chassis/enclosure user manual. First, inspect the cabinet. Every cabinet is different when it comes to compartment options. You should find a stack of compartments with different sizes somewhere inside the cabinet. They might have little plastic switches, in which case they're tool-less compartments, or they might just look like metal clamps. Storage generally comes in two sizes, 2.5-inch (HDD and SSD) and 3.5-inch (HDD). Most 3.5-inch bays can accept 2.5-inch drives, but not the other way around (some 3.5-inch bays will have trays that are not designed for 2.5-inch drives, but all modes can accommodate 2.5-inch bays). You may also see larger compartments in the cabinet. They are for larger drives like optical drives and are usually located at the front of the case, near the top. If you have tool-less bays, each bay will have its own plastic switch or lever. Open or unlock the lever or switch and you can remove the tray. Place the drive in the tray. Some 3.5-inch trays have been designed to accept 2.5-inch trays. If this is the case, you'll need to screw the 2.5-inch drive into the 3.5-inch tray so it doesn't move.

STEP 10: INSTALL THE OPERATING SYSTEM

Parts/Tools: PC, monitor, mouse, keyboard, OS saved on flash drive The first screen you see will tell you to press a key to enter system setup or BIOS. Press the key to open the BIOS. (If the screen flickers too quickly to see the key, check your motherboard user manual.) First of all, you should check if all the components are installed and recognized. Find the page in the BIOS that displays your PC's system information (different motherboards will have different BIOS settings, but you should be able to find a screen that gives you this information) and check to see if the system is recognizing everything you've installed so far. the moment. Next, browse the BIOS until you find the Boot page (could be called "Boot Order" or "Boot Priority"). Change the boot order so that the flash drive comes first and the drive you want to install the operating system on (if you're using an SSD as a boot drive, you'll need to install the operating system there) comes second. Restart the computer. The computer will boot from the USB drive and the operating system installer will appear. Follow the instructions to finish the installation.

IT DOESN'T END HERE

If you've managed to complete the entire guide, congratulations on finishing your build (especially if it's your first time)! The work, however, does not necessarily end here. The great thing about building your own gaming PC is that the task is never truly done. As advances in hardware continue to emerge, the ability of a custom PC to personalize it is virtually limitless. Your platform can be as up-to-date as you want, according to both your needs and your budget. The PC you just built will serve as the foundation for all the gaming experiences that lie ahead, and tweaking the components is part of the fun. Now that you know how to build a gaming PC, consider integrating your computer into a full battle station.
